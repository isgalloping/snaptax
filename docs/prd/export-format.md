# SnapTax 导出格式增强方案

作为一款不对接 IRS 电子申报的前置工具，SnapTax 的核心定位是**“完美喂饱各类报税软件/会计师的智能收据盒”**。为了让 1099 蓝领工人在税季实现“无缝复制粘贴”，除了目前支持的 **CPA (Excel/CSV)**、**TXF (Tax Exchange Format)** 和 **TurboTax CSV** 之外，强烈建议补充以下 3 种核心格式/交付物，彻底做死产品生态闭环：

---

## 1. 必做补充：官方 Schedule C 镜像 PDF（可视化对账单）
这是对蓝领工人最直观、最能带来“安全感”的格式。

* **用户痛点：** 很多独立报税工具（例如这两年极火、完全免费的 **FreeTaxUSA**）不支持导入第三方 CSV，但它们会按 Schedule C 的类目一门一门地引导用户手填。
* **产品实现：** 导出一个直观的 PDF，其类目排版、顺序甚至编号**完全像素级模仿官方 Schedule C 的 Part II (Expenses) 结构**（例如：Line 9 是 Car & Truck, Line 18 是 Office Expense）。
* **交付体验：** 用户把 PDF 打印出来或在手机上打开，对着免费报税软件，**看到 Line 9 填 Line 9，看到 Line 18 填 Line 18**。5 分钟无脑抄完，体验绝佳。

---

## 2. 格式升级：QuickBooks 兼容格式 (.QBO / .QIF / .OFX)
用于承接那些打算从 SnapTax 把数据迁移到更正规记账/财务生态的用户。

* **用户痛点：** 蓝领工人年底发现自己赚了大钱，决定不自己报了，要把一年的收据流水扔给传统 CPA（注册会计师）。而美国几乎所有老牌 CPA 必用的记账软件都是 Intuit 旗下的 QuickBooks。CPA 往往会要求：“把你的 QuickBooks 数据导给我。”
* **产品实现：** 支持导出 `.QBO` (QuickBooks Online) 或通用的 `.QIF / .OFX` 金融数据格式。这能让 SnapTax 瞬间具备与其他主流记账生态对话的能力。

---

## 3. 安全底牌：收据原图压缩包（Receipts ZIP Audit Trail）
这是对抗 IRS（美国国税局）审计（Audit）的终极防御大杀器。

* **用户痛点：** 1099 蓝领工人最怕被 IRS 抽查审计。哪怕在软件里导入了 CSV，如果 IRS 来查，用户必须拿出每一笔开销的原始小票（Receipt）证明，否则面临补税和罚款。
* **产品实现：** 当用户付费点击导出时，系统在云端生成一个打包好的 `receipts.zip`。
  * **关键细节：** 压缩包内部的图片命名必须与 CSV 报告中的条目一一对应（例如：`2025_02_14_HomeDepot_34.50.jpg`）。
  * **文案杀伤力：** *“为您导出整年抵税清单的同时，打包生成一份『IRS防审计收据凭证包』。一旦被抽查，把 ZIP 直接甩给审计员即可。”*

## 📋 最终 SnapTax 导出面板功能矩阵（建议）

为了保持首屏干净和“Industrial Rugged（工业硬朗）”的高对比度风格，你的导出下载页面可以做成极简的大按钮分类：

*   **📄 导出 IRS 规范 Schedule C 对账单 (PDF)**
    *   *适合全网免费报税工具（如 FreeTaxUSA），对着页面直接手填。*
*   **📊 导出 TurboTax 专用导入文件 (.TXF)**
    *   *适合 TurboTax / H&R Block 桌面版直接导入。*
*   **📥 导出 会计师标准对账表 (.CSV / Excel)**
    *   *适合直接发给自己的 CPA 会计师帮着报。*
*   **📦 下载 1099 防审计收据原图凭证包 (.ZIP)**
    *   *本地留底，专门应对 IRS 抽查审计。*
